## 🔧 开发者指南

### 添加新翻译
```typescript
// 在 lib/i18n.ts 的 fallbackTranslations 中添加
'new_key': {
  'en': 'English text',
  'zh': '中文文本',
  // ... 其他语言
}
```

### 在组件中使用
```tsx
const { t } = useTranslation('/')
const text = t('key.path', 'Default text')
```

### 添加新语言
1. 在 `SupportedLanguage` 类型中添加语言代码
2. 创建 `app/[lang_code]/page.tsx` 路由文件
3. 在翻译数据中添加对应翻译
4. 更新 `LanguageSwitcher` 组件
